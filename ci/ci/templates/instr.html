Инструкция работы в Vscode

Работа по протоколу sftp

Для работы по sftp необходимо установить соответствующий плагин.

После установки нажимаем комбинацию Ctrl+Shift+p и набираем sftp и далее пункт SFTP Config

Прописываем настройки и нажимаем иконку сбоку.

Для того, чтобы начать редактирование необходимо вызвать контекстное меню правой кнопкой мыши и выбрать Edit in Local.

Работа по протоколу ssh

Для работы по ssh необходимо установить соответствующий плагин.

После установки нажимаем зеленую иконку в левом нижнем углу и далее пункт Add new SSH Host

Вводим адрес сервера и порта.

Выбираем куда сохранить настройки.

Вводим пароль 123.

Выбираем Open Folder через верхнюю иконку.

Находим каталог backend.
